# SimpleJar
A lightweight easy minecraft launcher that let's you play minecraft with a minimal footprint. Built with python!

## Features 

**Multi account management** : lets you manage multiple accounts easily, as well as play in offline mode.
**Mod Support** : Download modpacks and mods using the modrinth api
**EXTREMELY fast** : blazing fast launcher that not only is light in size, but extremely fast and low on system resources.
